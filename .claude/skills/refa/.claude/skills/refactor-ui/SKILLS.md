# Skill: refactor-ui

Objetivo:
Padronizar identidade visual e UX do CRM sem quebrar layout ou integração.

Regras:
- usar a paleta oficial da marca
- padronizar sidebar, header, botões, cards, tabelas, inputs e modais
- criar variáveis CSS em :root
- manter visual corporativo premium
- não remover funcionalidades existentes
- sempre preservar responsividade
